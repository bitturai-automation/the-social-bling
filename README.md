# The Social Bling — Practice Project

> ⚠️ **This is a practice/demo project built for learning purposes only.**
> Not an official website. Created to practice HTML, CSS & JavaScript skills.

---

A fully responsive multi-page website I built as a **personal practice project** to sharpen my frontend development skills — clean dark UI, animations, and multi-page structure all done from scratch.

---

## 📁 Project Structure

```
the-social-bling/
├── index.html       # Home page — Hero, Services slider, Results, FAQ
├── about.html       # About page — Our story, Team, Philosophy
├── services.html    # Services page — Full service offerings
├── contact.html     # Contact page — Book a strategy call
└── README.md        # You're here
```

---

## ✨ What I Practiced

- **Dark luxury UI design** — Black backgrounds, white typography, glass morphism
- **Custom animated cursor** — Mix-blend-mode difference effect
- **GSAP scroll animations** — ScrollTrigger reveals and parallax
- **Services card slider** — Touch/drag enabled carousel
- **Floating animated stat cards** — Hero section metrics
- **Particle effects** — Ambient floating particles
- **FAQ accordion** — Smooth expand/collapse
- **Logo marquee** — Infinite scroll strip
- **Mobile hamburger menu** — Full-screen nav overlay
- **Multi-page structure** — 4 separate HTML pages with shared design system

---

## 🚀 How to Run

Just open `index.html` in your browser — no build tools needed.

Or deploy free on GitHub Pages:
1. Push to GitHub
2. **Settings → Pages → Deploy from `main` branch**
3. Live at: `https://yourusername.github.io/the-social-bling/`
